Google Photos Sync (gphotos-sync)

Google Photos Sync is a backup tool for your Google Photos cloud storage.

It downloads all photos and videos the user has uploaded to Google Photos. It also organizes the media in the local file system using album information. Additional Google Photos ‘Creations’ such as animations, panoramas, movies, effects and collages are also backed up.

PyPI

pip install gphotos-sync
